### 1. **Grantor/Grantee Extraction from Metadata**

**Source**: Search results HTML metadata (NOT from OCR)

**Flow**:
```
Eagle Assessor Search Results HTML
    ↓ (Parse <ul> elements with <b> labels)
    ↓
Extract grantor/grantee names as arrays
    ↓ (Normalize to pipe-separated strings)
    ↓
Store in database gila_leads.grantors, gila_leads.grantees
```

**Extraction Code** (`gila/extractor.py:325-420`):
- Function: `_extract_column_values()`
- Pattern: Finds `<ul class="selfServiceSearchResultColumn">` containers
- Labels: "Grantor" → "grantor", "Grantee" → "grantee"
- Values: Extracts bold text values `<b>NAME</b>`
- Result: Multiple names properly separated by " | "

**Example Data**:
```
grantors:  "NORRIS TIFFANIE M"
grantees:  "ARIZONA HEALTH CARE COST CONTAINMENT SYSTEM | ARIZONA LONG TERM CARE SYSTEM | RAWLINGS COMPANY, LLC"
```

### 2. **Document URL Access & Extraction**

**Source**: Detail page HTML (parsed from link in search results)

**Flow**:
```
Detail Page URL (from search results)
    ↓ (Fetch detail page via requests.Session)
    ↓
Parse HTML for PDF GUID pattern:
  /web/document-image-pdfjs/{doc_id}/{UUID}/document.pdf
    ↓
Construct full document URL with headers:
  https://selfservice.gilacountyaz.gov/web/document-image-pdfjs/
  {doc_id}/{uuid}/document.pdf?allowDownload=true&index=1
    ↓
Store in database gila_leads.image_urls
```

**Document URL Extraction Code** (`gila/extractor.py:698-746`):
- Function: `enrich_record_with_ocr()`
- UUID Pattern: `[a-f0-9]{8}-[a-f0-9]{4}-[a-f0-9]{4}-[a-f0-9]{4}-[a-f0-9]{12}`
- Regex: `/web/document-image-pdfjs/[^/]+/({UUID})/[^"'/?]+?\.pdf`
- Result: Full accessible URL with download parameters

**Example URL**:
```
https://selfservice.gilacountyaz.gov/web/document-image-pdfjs/
DOC2352S783/a1b2c3d4-e5f6-4789-0abc-def123456789/
document.pdf?allowDownload=true&index=1
```

### 3. **Database Storage & Idempotence**

**Table**: `gila_leads`

**Grantor/Grantee Columns**:
```sql
grantors         text,    -- "NAME1 | NAME2"
grantees         text,    -- "LENDER1 | LENDER2"
```

**Document URL Column**:
```sql
image_urls       text,    -- "https://selfservice.gilacountyaz.gov/..."
```

**Upsert Strategy** (`gila/run_gila_interval.py:240-320`):
```sql
INSERT INTO gila_leads (grantors, grantees, image_urls, ...)
ON CONFLICT (source_county, document_id)
DO UPDATE SET grantors = excluded.grantors,
             grantees = excluded.grantees,
             image_urls = excluded.image_urls
```

**Result**: Safe re-runs without duplicates, proper updates of enriched data
