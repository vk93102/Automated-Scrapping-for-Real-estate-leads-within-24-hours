```sql
grantors         text,   -- "GRANTOR1 | GRANTOR2"

trustor          text,   -- Can be derived from grantors

```

## 🔨 HOW IT WORKS (Technical)

**Extraction** (`conino/extractor.py`):
- Searches for `<ul class="selfServiceSearchResultColumn...">`
- Extracts `<li>` items with `<b>` bold text labels
- Labels like "grantor", "grantee", "trustor" etc.
- Returns as arrays of strings

**Storage** (`conino/run_conino_interval.py`):
- Converts arrays to pipe-separated strings
- Stores in `grantors` and `grantees` text columns
- Upserts by (source_county, document_id) for idempotence

**Retrieval**:
- Query `conino_leads` table
- Split on " | " if parsing individual names needed
