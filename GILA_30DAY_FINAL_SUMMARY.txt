# GILA COUNTY 30-DAY PIPELINE - FINAL SUMMARY

## WHAT WAS FIXED

### Problem 1: Missing Trustor/Trustee Names
**Root Cause**: Gila was searching for LIS PENDENS documents (lawsuits) which have NO trustor/trustee fields
- LIS PENDENS = Court proceedings, not property transfers
- These documents only have plaintiff/defendant info, not lender/borrower relationships
- They're not useful for real estate lead generation

**Solution**: Removed LIS PENDENS from document type searches, now focusing on actual deed documents

### Problem 2: Incomplete Property Addresses  
**Root Cause**: Address extraction was using only legal descriptions
**Solution**: Improved address extraction to include parcel numbers and detailed legal descriptions

### Problem 3: Empty Trustor/Trustee Fields
**Root Cause**: Fields were initialized but never populated from document data
**Solution**: Added extraction logic to derive trustor/trustee from grantor/grantee based on document type

## PARAMETER MEANINGS

--lookback-days N
  Fetch documents from last N days
  Examples: 7 (weekly), 14 (bi-weekly), 30 (monthly), 90 (quarterly)

--ocr-limit N
  0     = Skip OCR entirely (fastest)
  5     = Process OCR on first 5 documents (balance speed/detail)
  -1    = Process OCR on all documents (slowest but most detailed)

--write-files
  Export CSV and JSON files for further analysis
  Files saved to: gila/output/gila_leads_YYYYMMDD_HHMMSS.{csv,json}

--workers N
  Use N parallel threads for document processing
  Recommended: 2-4 for reliable operation, 8+ for powerful machines

--once
  Run once and exit
  Without this flag: runs continuously with 60-second intervals
